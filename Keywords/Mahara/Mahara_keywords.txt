*** Settings ***
Library     SeleniumLibrary

*** Keywords ***
Login to Mahara
    [Arguments]    ${username}    ${password}
    SeleniumLibrary.Input Text    id=login_login_username    ${username}
    SeleniumLibrary.Input Password    id=login_login_password    ${password}
    SeleniumLibrary.Click Button    Login    # value in unnecessary (it's more clean not to use it)   
    
Open browser to Mahara
    SeleniumLibrary.Open Browser    https://demo.mahara.org/    chrome
    SeleniumLibrary.Maximize Browser Window
     
    
 Capture Screenshot and Close Browser
     SeleniumLibrary.Capture Page Screenshot
     SeleniumLibrary.Close Browser
     
Mahara Valid Login
    [Arguments]    ${username}    ${password}    ${name}
    Open browser to Mahara
    Login to Mahara    ${username}    ${password}
    SeleniumLibrary.Wait Until Page Contains Element    https://demo.mahara.org/user/${name}
    SeleniumLibrary.Page Should Contain Element    xpath=//a[@class="user-icon"]    # /html/body/header/div/div/div[2]/a
    Capture Screenshot and Close Browser
